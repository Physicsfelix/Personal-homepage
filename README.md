## 本地预览与构建

安装 [Quarto](https://quarto.org/docs/get-started/) 后，在项目目录中运行：

```powershell
quarto preview
```

生成完整静态网站：

```powershell
quarto render
```

## 更新与发布

网站由 GitHub Actions 与 GitHub Pages 自动发布。更新源文件或获准公开的讲义后，应先在本地预览和渲染，确认页面与下载链接正常，再将修改推送到默认分支；`.github/workflows/publish.yml` 会构建网站并部署最新版本。
